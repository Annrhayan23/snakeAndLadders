

# Advanced Snake and Ladders Game (Java Swing)

A visually interactive **Snake and Ladders game** built using **Java Swing**. This project features a fully functional board, animated dice rolls, snakes and ladders effects, and turn-based two-player gameplay.

---

## Features

* **10×10 board** with numbered cells and zig-zag layout.
* **Two players** with colored tokens (Blue & Magenta).
* **Animated dice roll** with random number generation.
* **Snakes and ladders** visualized on the board with lines.
* **Pop-up messages** for ladders and snakes.
* **Win detection** and automatic game reset.
* Smooth **player token movement animation**.

---


## How to Run

1. Make sure you have **Java JDK 8 or above** installed.
2. Clone the repository:

```bash
git clone https://github.com/Annrhayan23/SnakeAndLadders.git
```

3. Navigate to the project folder:

```bash
cd SnakeAndLadders
```

4. Compile the Java program:

```bash
javac SnakeAndLadders.java
```

5. Run the game:

```bash
java SnakeAndLadders
```

---

## Controls

* Click the **“Roll Dice”** button to roll the dice and move your player.
* Players move alternately.
* Land on a ladder to climb up, or a snake to slide down.
* First player to reach **cell 100** wins.

---

## Technologies Used

* Java 17+ (Swing GUI)
* Java AWT for graphics and layout
* Random class for dice simulation
* HashMaps for storing snakes and ladders
